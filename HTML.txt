<audio src="midia/360ytmp3.com_64kbps-funny-meme-sounds.mp3" controls autoplay></audio>

Outra maneira de adicionar um audio no seu site é usando a TAG AUDIO mas excluindo o scr

<audio controls>
<source src="midia/360ytmp3.com_64kbps-funny-meme-sounds.mp3" type="audio/mpeg">

-------------Centralização Vertical de Caixas--------------------09/03/2024
para se centralizar uma caixa verticalmente deve se ter 2 coisas, um container como o article ou section e uma div para ser centralizada,apos ter isso 
o CONTAINER deve ter a TAG POSITION: RELATIVE (para poder mudar o tipo de posição da caixa) e na div (caixa) deve ser a TAG POSITION: ABSOLUTE (posição absoluta) assim desbloqueando os comando left e top
que se deve usar para centralizar a porcentagem de 50% então ficaria:

.container {
   background-color: black;
   position: relative (apesar de que a posição relativa ja vem normalmente em uma div)
}

.caixa {
   background-color: white;
   position: absolute (absoluta)
   left 50%;
   top 50%;
}

com isso a div deve estar centralizada,mas não totalmente. para centralizar ela totalmente existe outro comando que é a TAG TRANSFORM: TRANSLATE (-50%, -50%)
que com isso faz realmente ficar centralizado,quando positivo o valor ele vai para a direita e baixo, quando negativo vai para esquerda e cima.

então a tag ficaria:

.container {
   background-color: black;
   position: relative (apesar de que a posição relativa ja vem normalmente em uma div)
}

.caixa {
   background-color: white;
   position: absolute (absoluta)
   left 50%;
   top 50%;
   transform: translate (-50%, -50%)
}
TAG CSS POSITION : Absolute/relative; (muda o tipo de posição,caso seja a absoluta libera as tags left e top)
TAG CSS LEFT : 50% (move a caixa para 50% para a esquerda)
TAG CSS TOP  : 50% (move a caixa para 50% para cima)
TAG CSS TRANSFROM: TRANSLATE (-50%,-50%)| (transforma a caixa para mover +50% para esquerda e cima, os valores negativos representam : esquerda,cima e os positivos representam: direita,baixo)
