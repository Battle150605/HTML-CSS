-------------IMAGEM QUE SE ADAPTA SOZINHA------------------------17/02/2024
O codigo faz a imagem se adaptar para o tipo de tela, mudando o tamanho da imagem com a tag PICTURE, dentro da TAG PICTURE se deve colocar a TAG SOURDE MEDIA:TYPE para definir a midia que ira trocar,apos isso devemos trocar o min width para max width assim colocando um tamanho maximo para a tela
codigo exemplo:
 <picture>
    <source media="(max-width:750px )" srcset="Imagens/Pequeno.png" type="image/png">
    
    <source media="(max-width:1050px )" srcset="Imagens/Media.png" type="image/png">
    <img src="imagens/Grande.png" alt="Imagem Flexivel">
</picture>

--------------AUDIO NO SITE-------------------------------------17/02/2024
para colocar um audio deve ser usado a TAG AUDIO e dentro dela, colocar as TAGS CONTROLS e AUTOPLAY
exemplo:
<audio src="midia/360ytmp3.com_64kbps-funny-meme-sounds.mp3" controls autoplay></audio>

Os audios suportados nessa TAG são: MP3,WAV,OGG
////////////////////
Outra maneira de adicionar um audio no seu site é usando a TAG AUDIO mas excluindo o scr
entre a tag audio e a outra adicionar a TAG SOURCE SCR TYPE para adicionar o audio:
exemplo:

<audio controls>
<source src="midia/360ytmp3.com_64kbps-funny-meme-sounds.mp3" type="audio/mpeg">
<p>Seu Navegador nao suporta esse tipo de audio</p>
</audio>

audio/mpeg // MP3
audio/ogg  // OGG
audio/wav  // WAV
